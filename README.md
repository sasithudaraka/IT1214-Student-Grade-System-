# IT1214-Student-Grade-System


## Project Purpose

This project is a Java console-based application developed for managing student academic records. The system allows teachers to store student information digitally and perform basic record management operations.

## Features

* Add a new student
* Display all student records
* Search for a student using Student ID
* Calculate and display the average marks of all students

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Git & GitHub

## How to Run the Program

### Step 1: Compile the Java Files
javac GradeSystem.java


### Step 2: Run the Program
java GradeSystem


## Menu Options

1. Add Student
2. Display All Students
3. Search Student by ID
4. Calculate Average Marks
5. Exit
